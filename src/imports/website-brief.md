6. Design Direction
Visual Style: - Clean, corporate, structured - Deep Navy primary backgrounds - Gold accent for primary CTAs and highlights - Purple accent for secondary highlights, hover states, and icon accents - White or very light gray content sections for readability balance - Clear section separation using subtle dividers - Strong typography hierarchy (bold headline fonts, clean sans-serif body font)
Typography Recommendation: - Headings: Modern bold sans-serif (e.g., Montserrat, Poppins, or similar) - Body: Clean sans-serif (e.g., Open Sans, Lato, or similar)
Button Styling: - Primary: Gold background / Navy text - Secondary: Navy outline with gold hover fill - Subtle elevation on hover for interactivity
User Experience Requirements: - Fast loading - Fully mobile responsive - Intuitive navigation - Professional animation (subtle, not flashy)
Brand Positioning Goal: The website must reflect an organized, leadership-driven HR department capable of operating at executive standards.
